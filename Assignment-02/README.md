## Exercise Sheet 2 – Interpolation Methods and Stellar Data

Implements linear interpolation and natural cubic spline interpolation from scratch. Tests both methods on stellar physics data (mass, luminosity, radius, temperature, lifetime) in linear and log-log space, demonstrating that log-log interpolation better captures power-law relationships. Derives stellar main-sequence lifetimes from luminosity and estimates turn-off mass for a stellar population using the secant root-finding method.

**Libraries:** NumPy, SciPy, Matplotlib, Pandas, SymPy
